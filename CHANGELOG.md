# Changelog

## [0.1.0] - 2026-07-05

### Added
- HTML parser with DOM tree and node traversal
- Content extraction (links, images, headings, metadata, forms, tables)
- HTML to Markdown conversion
- HTML to plain text conversion
- HTML minification with configurable options
- HTML formatting with configurable indentation
- HTML validation (accessibility, security, SEO)
- Semantic HTML diffing
- CLI with 8 commands (extract, convert, minify, format, validate, diff, info, version)
- 65 tests passing
